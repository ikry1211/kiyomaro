## 3. 提案の方向性

### 3.1. 提案書の構成

Zoom打合せの「全体戦略を整理すべき」という合意を踏まえ、**全体像を見せつつLINEを先行実施**する構成で提案する。

```
提案書
├── 1. SNS全体戦略（全体像と優先順位）
│   → LINE最優先。Instagramは最低限の体裁。他は保留
│
├── 2. LINE運用（メイン提案・先行実施）
│   → 導線設計・料金・スケジュール
│
└── 3. Instagram運用（セット提案・LINE安定後に開始）
    → 方向性・料金・工数削減
```

> **見積もりは別々に出す。** 先方が「まずLINEだけ」と判断できる余地を残す。

### 3.2. 先方への提案フレーム

> 「全体としてはLINE → Instagramの優先順位を推奨します。
> まずLINEで申込フォーム離脱者の獲得を仕組み化し、広告のCVRを改善します。
> LINEの運用が安定したら、Instagramに着手していく形がリスクも少なく進められます。」

### 3.3. LINE運用の提案方針

**LINE公式アカウントの標準機能をベースに、弊社独自の運用ノウハウで高度な運用を行う。**（L-STEP不要）

> ⚠️ **SOCT内部メモ:** 裏側ではLLM（セグメント分析・メッセージ生成・配信最適化）を活用するが、先方には「AI」とは言わない。「弊社の運用ノウハウ」「独自の分析手法」として結果だけ見せる。

**先方への訴求ポイント：**

1. **「拡張ツールの月額費用がかからない」** — L-STEP不要。LINE公式アカウントの標準機能で運用
2. **「広告とLINEを直結した分析・最適化」** — CVR改善にダイレクトに効く
3. **「情報はすべてLINE公式アカウントに残る」** — ロックインなし
4. **「弊社独自の運用ノウハウでパーソナライズド配信」** — データに基づく最適化

> **フレーミング:** ツール比較にもAI訴求にも持ち込まない。「LINE公式アカウントを、弊社の運用ノウハウで最大限に活かします」という提案。

#### データ残存の設計方針

| 機能 | 使うツール | 理由 |
|:---|:---|:---|
| リッチメニュー | **管理画面で設定** | 管理画面に残る。クライアントが後からでも編集可能 |
| 手動チャット対応 | **管理画面のチャット** | 応対履歴が残る |
| タグ付け | **管理画面の手動タグ** | ユーザー属性が管理画面に残る |
| 通常の一斉配信 | **管理画面の配信** | 配信履歴・統計が管理画面に残る |
| ステップ配信（自動化） | **Messaging API** | 管理画面にこの機能がないため |
| 高度なセグメント配信 | **Messaging API** | 管理画面の条件設定では不十分な場合 |
| 分析レポート | **SOCT独自 → 月次で共有** | API側のデータもクライアントに蓄積される形で引き渡し |

### 3.4. Instagram提案のポイント

| ポイント | 内容 |
|:---|:---|
| **優先順位は2番目** | LINEが安定してから開始。提案書には含めるが「Phase 2」の位置づけ |
| **「数字を見ながら提案できる体制」を強調** | 広告 → LINE → Instagram を横断してKPIを見る体制 |
| **指名検索数をKPIに設定** | フォロワー数や直接CVではなく、ブランド認知の指標 |
| **動画制作の効率化** | 1日8時間で2本 → 大幅な工数削減を提案 |

### 4.2. 価格設計の方針

> **LLMで効率化した分は安売りの理由にしない。相場の下限で競争力のある価格を設定し、効率化した分はSOCTの粗利率改善に回す。**

| 考え方 | 内容 |
|:---|:---|
| **相場の下限で設定** | 実績がないため相場の上限は狙わない。ただし安すぎると信頼性を損なう |
| **LLM効率化 = 粗利改善** | 他社が月40〜80時間かける作業を月10〜15時間で回す。差分がSOCTの競争優位 |
| **工数に余裕を持たせる** | 安すぎてスコープが回らないリスクを排除 |

### 4.3. LINE運用 — 料金案

| フェーズ | 内容 | 料金 | 期間 |
|:---|:---|:---|:---|
| **初期構築** | LINE公式アカウント設計・ステップ配信構築・リッチメニュー・広告連携設定 | **¥100,000** | 初月 |
| **月額運用** | 配信設計・分析レポート・シナリオ改善・広告最適化連携 | **¥80,000/月** | 2ヶ月目〜 |

**料金の妥当性：**

| 観点 | 検証 |
|:---|:---|
| **vs 市場相場** | 標準運用型 ¥3〜10万の上位。広告連携込みなので戦略型（¥15万〜）の下にも位置する。**競争力あり** |
| **vs L-STEP自前運用** | L-STEPプロプラン ¥32,780/月 + 自社運用工数（月10〜20h）。SOCTに任せれば¥80,000/月で工数ゼロ。運用代行の人件費を考えると合理的 |
| **工数の余裕** | 吉岡さん月8〜12時間を確保可能。配信設計・分析・レポートに十分な時間を割ける |
| **SOCTの粗利** | ¥80,000 × 40% = ¥32,000/月。LLM活用で効率化した分が粗利率の改善に直結 |

> 実績ができたら戦略型の水準（¥100,000〜150,000）への移行を提案。
