## 2. 計算方法

```
月次価格 = 単価/人月 × 月稼働
合計価格 = Σ（月次価格 × 開発期間）
```

見積もりは**役務（プロジェクト管理・システム設計・実装等）**で区分する。担当者名は記載しない。

---

## 3. 標準的な役務区分

| 役務 | 内容 |
|:---|:---|
| **プロジェクト管理** | 技術方針・品質管理・意思決定 |
| **システム設計** | 要件定義・仕様策定・顧客折衝 |
| **プロトタイピング** | 画面設計・デザイン・プロトタイプ構築 |
| **実装** | フロント・バック・DB・インフラ |
| **テスト** | 結合テスト・不具合報告 |

---

## 4. 二層構造

| レイヤー | 用途 | 計算方法 |
|:---|:---|:---|
| **クライアント提示価格** | 見積書・提案書 | 単価テーブル × 月稼働 |
| **社内原価** | 利益管理（非開示） | 実際の外注費のみ（固定費・既存ツール費・クライアント環境費は除外） |

> クライアントには単価テーブルベースの価格を提示。AI-native開発による実コスト圧縮分が粗利になる。

## 6. クライアントへの提示ルール

### 6.1. 見せるもの / 見せないもの

| 対象 | 見せる | 見せない |
|:---|:---|:---|
| クライアント | 月額 × 期間、役務一覧、成果物 | 工数テーブル（ランク・月稼働） |
| 社内 | 全て | — |

### 6.2. 「内訳は？」への回答

月額に含まれる**役務（何をするか）** で説明する。**工数（何時間やるか）** では説明しない。

```
例: 月額¥1,280,000には以下の役務が含まれます:
・プロジェクト管理（技術方針・品質管理）
・システム設計（要件定義・仕様策定）
・プロトタイピング（画面設計・動くプロトタイプ）
・週1回のMTG・レビュー
```

### 6.3. 「何名体制？」への回答

```
少数精鋭チーム + AI技術のハイブリッド体制です。
従来型の5〜10名体制と同等のアウトプットを実現します。
```

### 6.4. 深掘りされた場合

「月額伴走型は時間ではなく**成果物で品質を担保する契約形態**です」に着地させる。

## 7. サービスメニュー（フェーズ制）

### 7.1. 3フェーズ体系

| Phase | 名称 | 月額 | 目的 | 成果物 |
|:---|:---|---:|:---|:---|
| **Phase 0** | プロトタイプ | **¥300,000** | 見せる | 動くデモ（主要画面3〜5画面）・技術提案書 |
| **Phase A** | 設計 | **¥800,000** | 決める | 要件定義書・画面設計・DB設計・詳細プロトタイプ |
| **Phase B** | 開発 | **¥1,300,000** | 作る | 本番アプリ・テスト済みリリース・運用引き渡し |

```
¥30万 → ¥80万 → ¥130万

「見て → 決めて → 作る」

どのフェーズからでも始められる。
どのフェーズでも止められる。
```

### 7.2. 各フェーズの原価構造（社内）

| Phase | 池田稼働 | 山田稼働 | 外注 | 原価/月 | 粗利率 |
|:---|:---|:---|:---|---:|---:|
| **0** | ヒアリング1h | — | — | **≒ ¥0** | **≒ 100%** |
| **A** | レビュー15% | PM代行 | — | **≒ ¥0** | **≒ 100%** |
| **B** | レビュー15% | テスト | 外注¥19万/月 | **¥192,000** | **85%** |

> 池田・山田は固定費のため原価に含まない。Phase 0・Aは実質原価ゼロ。

### 7.3. 案件への適用パターン

| パターン | 使い方 | 合計 | 向いている案件 |
|:---|:---|---:|:---|
| **0のみ** | PoC・提案 | ¥30万 | 予算未確定の新規営業 |
| **0 → A** | 検証 → 設計 | ¥30万 + ¥80万×N | 要件が不明確な案件 |
| **A → B** | 設計 → 本開発 | ¥80万×N + ¥130万×N | **標準パターン（中川様案件）** |
| **0 → A → B** | フルコース | ¥30万 + ¥80万×N + ¥130万×N | 初見クライアント |
| **Bのみ** | 開発のみ | ¥130万×N | 要件確定済みの案件 |

### 7.4. クライアントへの伝え方

```
「弊社では3つのフェーズをご用意しています。

 Phase 0（月¥30万）: まずプロトタイプをお見せします。
 Phase A（月¥80万）: 要件を固め、設計を詰めます。
 Phase B（月¥130万）: 本格的に開発し、リリースします。

 各フェーズ完了時に次に進むかご判断いただけます。
 まずはPhase 0から始めてみませんか？」
```
