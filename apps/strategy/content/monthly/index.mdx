# 📊 月次分析

毎月の経営状況を定点観測し、戦略の軌道修正を行うためのセクションです。

## 分析の構成

各月の分析レポートには以下を含みます：

| セクション | 内容 |
|:---|:---|
| 財務サマリー | 月次収益・キャッシュフロー・予実差分 |
| 事業別状況 | 各事業の進捗と健全性評価 |
| 組織・体制 | メンバーのアサインと稼働状況 |
| リスク評価 | リスク一覧と対策状況 |
| 次月アクション | 優先取り組み事項 |
