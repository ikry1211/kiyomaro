## 1. 事業概要

| 項目 | 内容 |
|:---|:---|
| プロダクト | 障害者雇用マッチング＆フォローアッププラットフォーム |
| コンセプト | **事業所と企業をつなぎ、採用からフォローアップまでを一気通貫で支援** |
| 課金モデル | **Phase 1-2は全機能無料 → Phase 3で成功報酬（¥10万/人を検討中）** |
| 位置づけ | はなえみブランドの主力サービス — Rakucoとは独立した収益軸 |

> **3/26 方針転換:** v1（月額サブスク）→ v2（フリーミアム）→ v3（LinkedIn型）→ **v4（先行無料→成功報酬）** に到達。「良いPFを作るほど課金されない」逆説を回避するため、採用成立を課金トリガーとするモデルに転換。詳細は[課金モデル仮説ドラフト v4](../plans/pricing-hypothesis)。

## 2. ターゲット市場

### 2.1. 供給サイド（事業所）

| セグメント | 規模 | 特徴 |
|:---|:---|:---|
| 就労移行支援 | 全国約3,300事業所 | 一般就労が目標。企業との関係構築済み |
| 就労継続A型 | 全国約4,400事業所 | 雇用契約あり。企業からの問い合わせ対応は少ない |
| 就労継続B型 | 全国約15,000事業所 | 最大市場。企業とのマッチングは未開拓 |

> **差別化ポイント:** 就労移行支援事業所だけでなく、**A型・B型の人材もカバー**できることが競合にない価値。

### 2.2. 需要サイド（企業）

| セグメント | 特徴 | ニーズ |
|:---|:---|:---|
| 40〜99名 | 2026/7〜新たに法定雇用率の対象。雇用義務1名〜 | 「誰をどう雇えばいい？」から支援が必要。初めてのケースが多い |
| 100〜299名 | 雇用率引き上げで追加採用が必要。雇用義務3〜8名 | リストベースでの効率的な候補者探し。既存ルート（HW）では限界 |
| 300名以上 | 納付金対象。特例子会社や専門部署があることも | 特定スキル保有者のスカウト。大量採用ニーズ |

> **ITスクール浜松ヒアリング確認済み:** 法定雇用率引き上げにより、従業員40〜120名規模の企業からの障害者雇用相談が増加中。

## 3. 市場規模（TAM / SAM / SOM）

> 需要サイド（企業課金）ベースで算出。法定雇用率2.5%（2026/7〜）により、従業員40名以上の企業が対象。

### 3.1. 対象企業数

| セグメント | 企業数 | 備考 |
|:---|---:|:---|
| 従業員40〜99名 | 約60,000社 | 2026/7〜新たに法定雇用率の対象 |
| 従業員100〜299名 | 約55,000社 | 追加採用ニーズが顕在化 |
| 従業員300名以上 | 約15,000社 | 特例子会社や専門部署あり |
| **合計** | **約130,000社** | — |

> データソース: 総務省「経済センサス」。全企業が障害者雇用にアクティブではないため、SAMで調整。

### 3.2. TAM / SAM / SOM

> 成功報酬モデル（¥10万/人）ベースで算出。年間採用件数は「対象企業数 × 企業あたり年間平均採用人数」。

| 指標 | 年間採用件数 | 年額（¥10万/人） | 定義 |
|:---|---:|---:|:---|
| **TAM**（全市場） | 約260,000人 | **¥260億** | 対象企業13万社 × 平均2人/年 |
| **SAM**（獲得可能市場） | 約10,000人 | **¥10億** | 積極採用企業5,000社 × 平均2人/年 |
| **SOM**（中期目標） | **120人** | **¥1,200万** | 2028年目標。月10件 × 12ヶ月。長野・静岡の複数拠点 |

> ⚠️ 成功報酬の金額（¥10万）は仮置き。β期間中のWTP測定で確定。企業あたり年間採用数は事業所ヒアリングベースの推定。

## 4. ビジネスモデル

### 4.1. 収益モデル（3軸）

```
┌──────────────────────────────────────────────────────────────────┐
│                はなえみ事業ポートフォリオ                       │
├──────────────────────────────────────────────────────────────────┤
│  Layer 1: PF直接収益                                           │
│  ┌──────────────────────────────┐                              │
│  │ ① マッチング収益              │                              │
│  │ PF経由の採用成立 → ¥10万/人   │                              │
│  │ （Phase 3以降）                │                              │
│  └──────────────────────────────┘                              │
├──────────────────────────────────────────────────────────────────┤
│  Layer 2: PF派生収益                                           │
│  ┌──────────────────────┬──────────────────────────┐           │
│  │ ② 定着支援連携        │ ③ コンサル・研修          │           │
│  │ なから実施→SOCT 50%   │ 株はなえみ実施→SOCT 50%   │           │
│  │ ¥15K/人・β期から即時  │ ¥250K〜/月・β期から即時   │           │
│  └──────────────────────┴──────────────────────────┘           │
└──────────────────────────────────────────────────────────────────┘
```

> ① はPFの直接収益（Phase 3以降）。②③ はPFのマッチング・企業接点から派生する間接収益で、β期間から即時に発生する。3軸を分けて管理し、§8で個別にシミュレーションする。

| 軸 | 収益源 | 主体 | 性質 | β期間の収益 |
|:---|:---|:---|:---|:---|
| **① マッチング** | 成功報酬 ¥10万/人 | SOCT（PF） | 直接収益 | ¥0（Phase 3以降） |
| **② 定着支援** | なから定着支援のSOCT取り分50% | SOCT（なから経由） | 連携収益 | **¥300K/月**（既存20名） |
| **③ コンサル** | PF経由新規コンサルの増分→SOCT取得 | SOCT（はなえみ経由） | 連携収益 | **¥100K/月**（推定） |

> **核心:** PF単体（①）はβ期間は収益ゼロ。②定着支援（¥300K）+ ③コンサル（¥100K）= **月¥400K**のPF派生収益でコスト（¥400K）をほぼカバー。
> ②③は同じ構造: 山田の稼働に伴い、連携先（なから/はなえみ）がサービスを実施。③ははなえみ売上¥800Kまでの増分100%、それ以降は50/50。


### 4.2. マッチングの2軸アプローチ

```
推薦型（事業所主導）⭐ 優先       スカウト型（企業主導）
────────────────              ────────────────
事業所が企業に利用者を推薦        企業がカタログを閲覧
 ↓                            ↓
推薦コメント付きで送信            「気になる」を送信
 ↓                            ↓
マッチング成立                  マッチング成立
 ↓                            ↓
正式エントリー（個人情報開示）    正式エントリー（個人情報開示）
 ↓                            ↓
選考・採用                     選考・採用
 ↓                            ↓
フォローアップフェーズへ          フォローアップフェーズへ
```

> **3/26 方針:** 推薦フローをスカウトフローより先に開発。事業所の現在の業務フローに沿った機能を優先し、無料で使ってもらう。

### 4.3. 定着支援ビジネスモデル（なからとの連携）

> ※ ここでの「定着支援」は、株式会社なからが行う就労定着支援業務を指す。はなえみダイレクトのプラットフォーム上の機能としては「フォローアップ管理」と呼称し区別する。

| 項目 | 内容 |
|:---|:---|
| 仕組み | マッチング後、半年間の利用者フォローを通じて定着支援業務を獲得 |
| 収益配分 | 定着支援収益の**50%をSOCT（はなえみダイレクト経由）**が受領 |
| 連携先 | **株式会社なから**（代表：北澤。はなえみが一部株式を保有） |
| ベース売上 | なから経由の既存20名を山田担当 → **¥1.5万/人 × 20人 = ¥30万/月** |
| スケーラビリティ | サビ管1人あたり60名の上限。規模拡大時はサビ管増員が必要 |

> PF経由のマッチングが増えるほど、定着支援の対象者も増える好循環。ただし、これはPFの収益ではなくなからの事業収益の一部。

### 4.4. 料金モデル検討ノート

> **ステータス:** ドラフト v4。β期間（7〜12月）の利用データで検証・確定。
> **方針:** 先行無料で市場を取り、PF定着後に成功報酬を導入する。
> **詳細:** 検討過程・競合調査・収益シミュレーションは別紙参照。

#### A. 仮説の変遷と結論

| Version | モデル | 棄却理由 |
|:---|:---|:---|
| v1 | 月額サブスク + フリーミアム | 推薦フロー全工程無料なら有料化トリガーが弱い |
| v2 | 両面フリーミアム | フォローアップ管理はロックインにならない |
| v3 | アクセス先制限（LinkedIn型） | 良いPFを作るほど課金されない逆説 |
| **v4** | **先行無料 → 成功報酬** | **現仮説** |

> 参考サービス14社（Wantedly, JobMedley, LITALICO, atGP, LinkedIn, Chatwork, Slack, Zoom, Notion, Canva, クラウドサイン等）の調査に基づく。

#### B. フェーズ別戦略

| Phase | 時期 | 課金 | コスト原資 |
|:---|:---|:---|:---|
| **1. β** | 2026/7〜12 | 全機能無料 | SOCT Rakucoチーム兼務 |
| **2. 拡大** | 2027〜 | 全機能無料 | 株式会社はなえみのコンサル収益 + SOCT Rakuco売上 |
| **3. マネタイズ** | PF定着後 | **成功報酬 ¥10万/人**（仮） | PF自体で収益化 |

#### C. バイラルループ設計

```
推薦フロー（事業所 → 企業）のバイラルループ:

① 事業所が利用者を登録 → AI自動プロフィール生成
② 事業所が既知の企業に推薦を送信
                                 ────── 企業側 ──────
                                 ③ メール通知で推薦を受信
                                    ↓ アカウント不要で閲覧可能
                                 ④ AIプロフィール + 推薦コメントを確認
                                    ↓ 興味あり
                                 ⑤ 無料アカウント作成 → チャット・選考
                                    ↓ 採用成功
                                 ⑥ Phase 3以降: 成功報酬が発生
```

> **導入の起点は事業所。** 事業所がAI自動入力の便利さからPFを使い始め、企業はPFで推薦を受け取るしかない（クラウドサイン型：送る側が選ぶ）。

| 3者チャット | ✅ 無料 | ✅ 無料 | — |
| 選考管理 | ✅ 無料 | ✅ 無料 | — |
| フォローアップ管理 | ✅ 無料 | ✅ 無料 | — |
| 求人登録・カタログ閲覧 | — | ✅ 無料 | — |

#### E. 価格の妥当性（年収¥180万の場合）

| チャネル | 金額 | はなえみダイレクトとの比較 |
|:---|---:|:---|
| ハローワーク | ¥0 | 無料だが手間がかかる。選べない |
| **はなえみダイレクト** | **¥10万** | **人材紹介の1/4〜1/6** |
| JobMedley | ¥4〜7.2万 | はなえみの0.4〜0.7倍 |
| LITALICO仕事ナビ | ¥18万 | はなえみの1.8倍 |
| 人材紹介 | ¥36〜54万 | はなえみの4〜5倍 |

> **¥10万 = 年収の5.6%。** LITALICOの半額、人材紹介の1/4〜1/6。β期間中にVan Westendorp法でWTPを測定して確定。

#### F. 「既存フロー置換」問題への回答

> **懸念:** 従来の電話紹介は無料。成功報酬を取ると電話に戻るのでは？
> **回答:** 企業が「PFを使うかどうか」を選ぶのではない。**事業所が選ぶ。** 事業所がPFで推薦を送れば、企業はPFで受け取るしかない。成功報酬は採用後に発生し、その頃にはPFの価値を実感している。

#### G. β期間中の検証ポイント

| # | 検証項目 | 方法 | 判断基準 |
|:---|:---|:---|:---|
| 1 | 推薦→企業アカウント作成率 | 推薦送信数 vs アカウント作成数 | 30%以上 |
| 2 | PF利用の継続率 | 月次アクティブ率 | 50%以上 |
| 3 | 事業所の業務効率改善 | 1推薦あたりの所要時間比較 | 電話比で50%以下 |
| 4 | 成功報酬のWTP | Van Westendorp法 + 金額別直接比較 | Q2〜Q3交差点が¥8〜12万 |
| 5 | PF外での採用完結リスク | PF経由の推薦→PF外で面接・採用の件数 | 抜け穴の深刻度を測定 |
| 6 | 企業の満足度 | NPS / ヒアリング | +30以上 |

> **最終判断のタイミング:** 2026年Q4（β終了時）。利用データ + 最低10社の企業ヒアリングで確定。

### 8.1. はなえみダイレクト（PF）の収益計画

> PF自体の直接収益。成功報酬のみ。

| 前提 | 値 | 根拠 |
|:---|---:|:---|
| 成功報酬 | ¥100,000/人 | β期間中にWTP測定で確定 |
| 月間固定コスト | ¥400K | §10の積上げ: COGS¥40K + 山田¥238K + 開発¥75K + 出張¥50K |

### 8.3. コンサル・研修収益（③コンサル — 株式会社はなえみ経由）

#### 収益分配モデル（案）

> 山田のPF営業兼務によるコンサル案件の新規獲得分をSOCTの収益とする。はなえみ売上が現状の2倍（¥800K/月）に達するまでは**増分100%をSOCT**、到達後は**50/50**。

```
はなえみ月間売上         SOCT取り分
──────────────────────────────────────
  ¥400K（現状）         ¥0（トントンのため余剰なし）
   ↓ 増分100% SOCT
  ¥800K                最大¥400K/月（増分全額）
   ↓ 超過分50/50
  ¥800K超              ¥400K +（超過分 × 50%）
```

|:---|---:|:---|

> ⚠️ 協業の方針はほぼ確定。正式合意後に分配モデルを確定する。
