### 2.4 会話内容の質的分析（全97件の通話読み取り）

各メンバーの代表的な通話（計97件）を直接読み、以下5軸でメンバー別の**会話傾向**を整理した。

#### A. アプローチ（導入〜目的伝達）

| メンバー | 傾向 |
|:---|:---|
| **野島** | 「一年前にもお電話した」と**過去の接点**から入る。時には「昨年5月ごろ」と時期を具体的に提示し、相手の記憶を呼び起こす。用件は簡潔に伝え、すぐに相手の反応を待つ |
| **日高** | 「以前システムのご案内をさせていただいた」と前回の内容を引用しつつ、早い段階で「バージョンアップした」「切り替えが増えている」と**変化のニュース**を伝える |
| **鍵谷** | 「国保連の請求システムの件で」と最も簡潔。すぐ「管理者の方いらっしゃいますか」と担当者確認に移行。**余計な前置きがなく、テンポが速い** |
| **藤本** | 「ラクコ」と**サービス名**を明確に使う唯一のメンバー。「就労支援に特化したラクコがバージョンアップした」と自社の立ち位置を早期に明示。過去の会話内容も正確に引用 |
| **福田** | 「国保連の請求システムに関する件で」が完全な定型。担当者に繋がると「一年前にもお電話しました→就労支援に特化してバージョンアップ→比較だけでも」と**全てを1発話で一気に話す（85文字/発話）** |
| **川上** | 「国保連の請求システムについてお尋ねしたかった」と**質問ベース**で入るため柔らかい印象だが、具体的な目的が伝わりにくい。用件説明に時間がかかり、相手が混乱するケースあり |

#### B. ヒアリング（相手状況の把握）

| メンバー | 傾向 |
|:---|:---|
| **野島** | 「そうだったんですね」で受容しつつ、「今もシステムは変わらず？」「お困りごとは？」と**段階的に深掘り**する。相手が話しやすい空気を作り、結果として平均29.8ターンの長い会話が生まれる |
| **日高** | 「ちなみに今は何をお使いですか？」「月額いくらぐらいですか？」「事業形態はA型ですか？」と**多角的に質問**。3.8問/通話はチーム最多。状況を把握してから説明に入るため、的を射た提案が可能 |
| **鍵谷** | 「すでにシステムは導入されていますか？」の一言がメイン。**質問はシンプルだが、回答に応じて提案を切り替える**判断力がある（アポ獲得通話では「ちなみにどちらのシステムを？」と深掘りに移行） |
| **藤本** | 「前回はノービーさん使ってたけど、その後お変わりないですか？」と前回情報を織り込んだ質問ができる。ただし質問後に相手の回答を十分聞く前に**提案に移行しがち** |
| **福田** | ヒアリングが最も弱い。定型質問51%と高く、相手の状況把握が浅いまま説明に入る。ランニングコストについて聞く試みはあるが、回答を活かした提案に繋がっていない |
| **川上** | 1.8問/通話で最少。事業形態の確認はするが、現状のシステムや不満点を深く聞く前に自社説明に入ってしまう傾向 |

#### C. 提案（サービス紹介の仕方）

| メンバー | 傾向 |
|:---|:---|
| **野島** | 機能説明にはほぼ踏み込まず、「ご比較だけでも」「Zoomで30分だけ」と**デモ誘導に絞った提案**。詳細説明はデモに委ねるという設計。CUSTOMERが「変えるつもりはない」と言っても、ランニングコストの話題で切り口を変える |
| **日高** | 「支援記録・勤怠管理・請求の3つができて月額19,800円」と**機能+価格の具体的なワンライナー**が武器。断られた後でもこのフレーズで最後のフックを残す。少ない通話で成果を出す効率の背景 |
| **鍵谷** | 「月々の費用が最大40%抑えられた事例もある」と**コスト削減訴求**が中心。資料送付への切り替えも提案でき、相手の温度に合わせた柔軟さがある |
| **藤本** | 「6ヶ月無料キャンペーン」「月額19,800円・縛りなし」と最も情報量が多い提案。ただし、相手の状況把握前に提示するため、「今のところ大丈夫です」と受け止められるケースも多い |
| **福田** | 「今の請求方法とのご比較だけでも」が定型。**長文の中にZoom提案が埋もれてしまう**。コスト削減の具体例を出す試み（「40〜50%抑えられた」）はあるが、相手が聞く体勢になっていないタイミングで出しがち |
| **川上** | 「請求ミス・実地指導・料金面の不安が解消された」と**課題解決型**の表現力はチーム随一。ただし断られると深追いせず、提案の引き出しが少ない |

#### D. 断り対応

| メンバー | 傾向 |
|:---|:---|
| **野島（83%継続）** | 「そうだったんですね」で一度受容 → 「ちなみに今のシステムでお困りの点は？」と別角度の質問 → 「4月以降改めさせていただく」と次アクション提示。**受容→質問→リカバリーの3ステップが自然** |
| **日高（80%継続）** | 断られても「他社様からの切り替えが増えてて」「月額19,800円で」と具体的な情報を1つだけ追加。**引き際でも有益な情報を残す**ことで次回の架電根拠を作る |
| **鍵谷（57%継続）** | 断られると「メールで資料を送らせていただく」への切り替えがスムーズ。ただし資料送付で終わるケースが多く、**Zoom誘導への再チャレンジが少ない** |
| **藤本（64%継続）** | 「年度末でお忙しいですよね。4月上旬とかだったら？」と時期を変えた再提案ができる。ただし、相手が明確に断った後も繰り返し提案するため、**CUSTOMER感情悪化率73%**の要因になっている |
| **福田（75%継続）** | 粘り強さは一定あるが、「今のご請求方法とのご比較だけでも→資料だけでも→メールアドレスを」と**段階的に要求を下げる**パターンが多い。本部の電話番号を聞き出す粘りもある一方、**毎回ほぼ同じフレーズの繰り返しで新規性がない** |
| **川上（42%継続）** | 「あーそうなんですね。かしこまりました」で即終了が多い。**断り後に会話を続ける引き出しがなく、チーム最低の継続率**。ただし「無料のものもある」など別切り口を出すケースも稀にあり |

#### E. クロージング（次アクションへの誘導）

| メンバー | 傾向 |
|:---|:---|
| **野島** | Zoom誘導は控えめ（3%）だが、「また4月以降に改めさせていただく」「担当者のお名前だけ伺っても」と**次回架電の布石を確実に打つ**。名前・連絡先を取得して撤退する型が多い |
| **日高（12%）** | 「来週か再来週で30分ほど、具体的にZoomでどうですか」と**その場で日程を確定しにいく**。チーム唯一の高クロージング率。断られても「午前午後どちらが都合良いですか」と二者択一で畳みかける |
| **鍵谷** | 「資料をお送りさせていただきます」が着地点。**メールアドレスの取得率が高い**ため、資料送付→フォロー架電のルートが確立されている |
| **藤本** | 「来週あたり30分お時間いただけないでしょうか」とZoom誘導するが、断られた後は「また時期改めて」で撤退。**名前と連絡先の取得は確実に行う** |
| **福田** | 「ぜひお時間の方よろしくお願いいたします」と曖昧なクロージング。日程確定に踏み込まず、**相手に委ねて終わる**ケースが多い |
| **川上** | 折り返し連絡先を伝えるケースがあり（チーム内では珍しい）。ただし、**能動的に日程を提案するクロージングがほぼない** |

|:---|:---|:---|:---|
| **A. アプローチ** | 過去接点を具体的に引用し、相手の記憶に訴える | 鍵谷は簡潔。藤本はサービス名+条件提示 | 福田は一方的な長文。川上は具体性不足 |

- **断り対応:** 「4月上旬ならどうですか」と時期変更の再提案ができるが、明確に断られた後も繰り返すため**感情悪化73%**の要因に

## 5. 定量 × 質的分析から導く仮説

定量比較（セクション2.1〜2.3, 2.5）と会話内容の質的分析（セクション2.4）を組み合わせ、**成果の差を生むメカニズムの仮説**を以下に整理する。

### 仮説1: 「ヒアリング先行型」は感情悪化を抑え、成果に繋がる

| 根拠（定量） | 根拠（質的） |
|:---|:---|
| 日高の感情改善率23%（チーム1位） / 悪化率31%（チーム最低） | 日高は「ちなみに今は何をお使いですか？」と質問→状況把握→提案の順序を徹底 |
| 藤本の感情悪化率73%（チーム最高） | 藤本は過去情報の引用力があるが、**相手の回答を待たず提案に移行**する傾向 |
| 日高5.8h/件 vs 藤本16.5h/件 | 両者とも価格・機能の情報量は豊富だが、**提案の順序が異なる** |

> **仮説:** 相手の状況を先に聞いてから説明する「**ヒアリング先行型**」は、感情悪化を抑えて信頼を得やすく、結果としてアポ獲得効率が高い。情報量自体よりも**出す順番**が成果を左右している。

### 仮説2: 「受容→別角度→布石」の断り対応が会話を伸ばし、成果を生む

| 根拠（定量） | 根拠（質的） |
|:---|:---|
| 野島の継続率83%（1位） / 平均29.8ターン（1位） / アポ25件（1位） | 野島は断りに「そうだったんですね」で受容し、**別角度の質問**で会話を続ける |
| 川上の継続率42%（最低） / アポ2件 | 川上は断られると「わかりました」で即終了。**引き出しがない** |
| 福田の継続率75%だが19.0h/件 | 福田は粘るが**同じフレーズの繰り返し**で、会話は続いても質が上がらない |

> **仮説:** 断り後の粘り方には**「粘りの質」**がある。野島のように受容→別角度で質問→次回布石という3ステップがあると会話が自然に伸びる。一方、福田のように同じ定型文を繰り返す粘り方は、継続率は高くても成果に繋がりにくい。

### 仮説3: 1発話の長さはCUSTOMERの参加度に影響する

| 根拠（定量） | 根拠（質的） |
|:---|:---|
| 福田85文字/発話（チーム平均の2倍）/ 19.0h/件 | 福田は「導入経緯→バージョンアップ→比較提案→Zoom誘導」を**1発話に全て詰め込む**。CUSTOMERの「はい」が連続し、参加感がない |
| 野島33文字 / 日高47文字 | 野島は短い発話で相手の反応を挟み、**会話のキャッチボール**を生む |

> **仮説:** 1発話33〜47文字が**対話型テレアポの適正レンジ**。これを大幅に超える（福田85文字）と相手が受動的になり、機会を逃す。逆に短すぎる（鍵谷28文字）と提案の説得力が落ちるが、テンポの良さで補える場合がある。

### 仮説4: クロージングの型は1つではない — 「その場確定型」と「布石型」
