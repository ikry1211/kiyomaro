# 動画制作自動化パイプライン

> **ステータス:** 企画段階
> **最終更新:** 2026-04-10
> **関連:** [マーケティング事業計画](/business/marketing) — Layer 2（AI活用型サービス）の実装手段 | [YouTube長尺動画 自動編集パイプライン](/plans/youtube-automation) — 長尺動画版

---

## 1. 背景と目的

### 1.1. 発端

既存の広告運用クライアントから「インスタコンサル + 動画編集」の相談を受けた。LP制作と同様に、AI活用 + 業務委託で効率化できる領域と判断。

### 1.2. 目的

クライアントのインスタリール動画制作を、**Remotion（React動画フレームワーク）+ LLM**で自動化し、以下を実現する。

| 目標 | 内容 |
|:---|:---|
| **制作コストの最小化** | 業務委託に出さず、ほぼ自動で動画を生成 |
| **品質の均一化** | テンプレートベースで受託品質を担保 |
| **スケーラビリティ** | クライアントが増えてもコストが線形に増えない |
| **Layer 2の差別化** | 「AI動画自動生成」をサービスメニューに組み込む |

## 2. 技術構成

### 2.1. 中核ツール: Remotion

| 項目 | 内容 |
|:---|:---|
| **概要** | React/TypeScriptで動画を構成し、プログラマティックにレンダリングするフレームワーク |
| **公式サイト** | https://remotion.dev |
| **ライセンス** | Company License: **$25/月/席**（Creatorsプラン） |
| **選定理由** | React/TSXがそのまま使える。CSS/flexboxでデザイン可能。GUIプレビュー（Studio）あり。エコシステム最大 |

#### 比較検討した代替ツール

| ツール | ライセンス | 棄却理由 |
|:---|:---|:---|
| **FFmpeg + Node.js** | 無料 | テロップのデザイン・アニメーションが貧弱。受託品質を出しづらい |
| **Revideo** | MIT（無料） | 独自API学習が必要。ドキュメント・実績が少ない。React知識を活かせない |
| **Motion Canvas** | MIT（無料） | 素材動画の読み込みが弱い。アニメーション特化で用途が違う |
| **Creatomate API** | $18/月〜 | APIに依存。自由度が低い |

### 2.2. 自動化パイプライン

```
┌─────────────────────────────────────────────────────────────┐
│  入力                                                        │
│  ├─ 素材動画（クライアントがスマホ撮影 → アップロード）          │
│  ├─ ブランド設定（ロゴ、フォント、カラー）── 初回のみ設定       │
│  └─ 企画テキスト（任意。なくても音声から自動生成）              │
│                                                              │
│  処理（自動化）                                               │
│  ① Whisper API → 音声からテロップテキスト自動生成              │
│  ② LLM → テロップをキャッチーに編集 + 構成提案                │
│  ③ LLM → Remotionコンポーネントを自動生成                     │
│  ④ Remotion → テンプレートに流し込み + レンダリング            │
│                                                              │
│  出力                                                        │
│  └─ 完成リール動画（1080×1920, 9:16, MP4）                   │
└─────────────────────────────────────────────────────────────┘
```

### 2.3. 各工程のLLM活用

| 工程 | 活用方法 | ツール |
|:---|:---|:---|
| **企画・台本** | 業種・目的を入力 → 台本パターンを自動生成 | Claude / ChatGPT |
| **テロップ文言** | 素材の概要 → キャッチーなテロップ文を生成 | Claude / ChatGPT |
| **字幕生成** | 音声 → テキスト自動変換 → テロップデータ（JSON） | **Whisper API** |
| **コード生成** | テロップデータ → Remotionコンポーネントを自動生成 | LLM（[Remotion公式プロンプト集](https://remotion.dev/prompts)活用） |
| **サムネ・カバー画像** | プロンプトから直接生成 | 画像生成AI |

## 3. 制作ワークフロー

### 3.1. クライアント向けフロー

```
① 吉岡さんがインスタ運用戦略を設計（コンサル）
  ↓
② クライアントがスマホで素材を撮影（撮影指示書ベース）
  ↓
③ 素材をアップロード
  ↓
④ パイプラインが自動処理（Whisper → LLM → Remotion）
  ↓
⑤ 吉岡さんが最終チェック・微調整
  ↓
⑥ 完成動画を納品
```

### 3.2. 工数見積もり（自動化後）

| 工程 | 担当 | 1本あたり |
|:---|:---|:---|
| 戦略・企画（月次） | 吉岡さん | 月1〜2時間 |
| 撮影指示書作成 | 吉岡さん or LLM | 10分 |
| 自動処理 | パイプライン | **数分（自動）** |
| 最終チェック | 吉岡さん | 5〜10分 |
| **1本あたり合計** | — | **約15〜20分** |

### 3.3. コスト構造

| 項目 | 月額 |
|:---|:---|
| Remotion Creatorsライセンス | **¥3,750**（$25/月） |
| Whisper API（音声文字起こし） | **¥100〜500**（利用量に応じて） |
| LLM API（テロップ・コード生成） | **¥500〜2,000** |
| **合計** | **¥4,350〜6,250/月** |

> クライアントへの請求（月¥5〜15万）に対し、ツールコストは**5%以下**。粗利率95%超の構造。

## 4. 動画の品質レベル

### 4.1. Remotionで実現できるクオリティ

| 要素 | 品質 | 備考 |
|:---|:---|:---|
| テロップデザイン | **◎** | CSS/Reactで自由自在。影・縁取り・グラデ・アニメ全対応 |
| トランジション | **◎** | Reactコンポーネントで表現可能 |
| テロップアニメーション | **◎** | CapCut並のポップアップ・ハイライトが実装可能 |
| エンコード品質 | **◎** | FFmpegベース。業界最高水準 |
| ブランド一貫性 | **◎** | テンプレートで色・フォント・ロゴを統一管理 |

### 4.2. 本番利用実績

Remotionは以下のプロダクションで使用されており、品質は十分に実証済み。

- **GitHub Unwrapped** — GitHubの年次レビュー動画
- **Submagic** — AI字幕付きショート動画生成サービス
- **Crayo.ai** — AIショート動画生成プラットフォーム
- **Revid.ai** — 自動ストーリーテリング

## 5. 段階的な導入計画

| フェーズ | 内容 | タイミング |
|:---|:---|:---|
| **Phase 0: テスト制作** | 業務委託 + CapCutで1〜2本テスト納品。案件を獲得 | Zoom後すぐ |
| **Phase 1: Remotionテンプレート構築** | 基本テンプレート（テロップ+BGM+ロゴ）を池田が構築 | 案件獲得後（1〜2日） |
| **Phase 2: Whisper連携** | 音声→テロップ自動生成パイプラインを構築 | Phase 1完了後 |
| **Phase 3: LLMフル連携** | 素材動画投入→完成動画出力の完全自動化 | Phase 2検証後 |

> **Phase 0は業務委託で始める**（実績なしリスクのカバー）。案件が取れたらPhase 1〜3で自動化に移行。
