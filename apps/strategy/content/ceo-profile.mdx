# CEOプロファイル — 池田涼

> **最終更新:** 2026-03-22
> **分析手法:** DISC行動特性 + Kolbe行動本能 + リスクプロファイル（観察ベース推定）

---

## 1. DISC行動特性（推定）

| 特性 | スコア | 説明 |
|:---|:---|:---|
| **D（主導型）** | **◉◉◉◉◯ 高** | 結果志向。無駄を嫌い、意思決定が速い。率直なコミュニケーション |
| **I（感化型）** | ◉◉◯◯◯ 低〜中 | 必要に応じて人を巻き込むが、関係構築より成果が優先 |
| **S（安定型）** | ◉◉◯◯◯ 低 | 現状維持より変化を選ぶ。「9ヶ月早く動ける」を¥2,500万より重視 |
| **C（慎重型）** | **◉◉◉◉◯ 高** | データで裏付けを取る。数字なしの判断は嫌う。ただし分析に時間をかけすぎない |

> **DC型（指揮官タイプ）**: データに基づき迅速に意思決定し、実行に移す。議論の無駄を排除し本質に集中する。

## 3. リスクプロファイル

### 3.1. 基本特性

| 項目 | 傾向 |
|:---|:---|
| **基本スタンス** | 守りを固めてから攻める。倒産防止共済¥720万、防衛ラインの設計が先行 |
| **リスク許容度** | 中〜高。ダウンサイドが限定されていると確認できれば大胆に動く |
| **撤退ライン** | 明確に設定する（例: 売却下限¥1億）。曖昧なまま進めない |
| **判断の基準** | 「最悪でもこうなる」の確認ができれば即決。月次プラマイゼロを確認した瞬間に判断が加速 |
| **有利時の傾向** | リードを取ると集中力が緩む傾向がある（テニスでも5-1→5-4になるタイプ）。ただし最終的には勝ち切る |

### 3.2. 隠れコストへの感度

| 項目 | 説明 |
|:---|:---|
| **メンタルバンドウィドス** | Rakucoの実務リソースは少ないが「関心を持っていかれる」コストを認識している。心配事が頭の片隅にある状態は、新規事業の創造的判断に使うべきリソースを常に消費する |
| **判断すべき総量** | 判断ポイントの多さ自体がリスク。事業売却は利益確定であると同時に「認知負荷のカット」 |
| **オプショナリティ** | 利益を確定してからフットワーク軽く複数の新規事業にトライする方を好む。全財産を1つに賭けるタイプではない |

### 3.3. ライフステージの影響

> 2025年11月に第一子が誕生。これによりリスク許容度が構造的に低下。

- 個人保証¥2,560万のリスクは、独身のときと子供がいるときでは意味が全く異なる
- 「自分一人なら最悪なんとかなる」が通用しなくなった
- **家族の安全という新しい判断軸**が加わり、ダウンサイドの許容範囲が狭くなっている

## 7. 過去の教訓

### 7.1.「安全圏＝気が緩む」パターン

> Innojin社の年間約¥5,000万の受託案件があった時期、Rakucoの営業体制改善などは継続しつつも、**新規事業開発への集中度が大きく低下した期間が約2年間あった。** 受託拡大のためにレディくるに¥1,000万弱を投じたが、担当の舟瀬さんに丸投げし1件も成果が出なかった。

**認識すべきリスク:** Rakuco売却でキャッシュ¥7,000万超を手にすると、同様のパターンが再発する可能性がある。ただし、当時と異なる緩和要因も存在する：

- **子供の存在** — 家族への責任感が行動のアンカーになる
- **AIによる事業開発意欲** — 技術的に面白いテーマがあること自体がモチベーション源
- **過去の学び** — 本人がこのパターンを明確に認識しており、再発を防ぐ意識がある

完全に緩む可能性は低いが、**集中度が下がる期間が生じるリスク**は残る。

> **重要な補足:** 「ずっと緊張していた方がいい」わけではない。キャッシュを得て安心すること自体は、不安ドリブンの行動から好奇心ドリブンの行動への移行であり、判断の質が上がる。注意すべきは「安心」と「無関心」の境界線のみ。

### 7.2. 委任の失敗パターン

| パターン | 詳細 |
|:---|:---|
| **過大評価バイアス** | 「自分がサボりたい」動機で人を過大評価する傾向を本人が認識。舟瀬さんへの委任がその典型 |
| **丸投げ vs 委任の違い** | 石丸さん（営業）やCSチームへの委任は機能している。違いは「数字を見ているか」。レディくるは数字を見ていなかった |

### 7.3. 本人の結論

> もっと直接コミットして新規事業の立ち上げを行うべきだった。

Rakucoも、池田さんが直接関与するようになってから成功した。**初期段階の事業は池田さんの直接コミットが必須。**

## 8. ビジョンと原動力

### 8.1. 5年後のビジョン

> 会社の規模にはこだわっていない。自分が自由に前向きな仕事に取り組めること、家族と過ごせることを重視。

- **スケール志向ではない。ライフスタイル × 自律性の確保が最優先**
- どこまで行っても中小企業のオーナーであることを受け入れている
- 「自由に前向きな仕事」がキーワード。やりたくない仕事を義務的にやる状態が最大のストレス源

### 8.2. エネルギーのソース

> 楽しさの源泉は**問題解決**。「もっとこうなればいいのに」という不満や理想主義的な気持ちが原動力でプロダクトを作っている。

- **課題ドリブン型の起業家。** 市場規模やトレンドではなく、自分が感じる「不」から事業が始まる
- ただし現時点では金銭的な不安も動機に含まれる。不安が解消されれば、より純粋に問題解決に集中できる
- Rakuco売却後は「不安の解消」と「問題解決への集中」の両方が実現する
- **不安ドリブンの行動と好奇心ドリブンの行動では質が違う。** 安心は弛緩ではなく、最適なコンディションへの移行

## 9. ストレスと意思決定

### 9.1. ストレス特性

| 項目 | 傾向 |
|:---|:---|
| **精神的耐性** | 高い。月次赤字・個人保証・複数事業の同時運営を冷静に処理できる |
| **身体的シグナル** | ストレスが体調に出やすい。精神が持っていても身体が先にアラートを出す |
| **波の存在** | 楽観的な日と悲観的な日がある。体調に左右される。本人も断定はできない |
| **基本姿勢** | ストレスを「耐える」のではなく**「構造的に消す」**。防衛ライン・倒産防止共済・経営ドキュメント整備はすべてストレス源の可視化と除去 |

### 9.2. 根底にある懸念

> 実務スキルが高い自負はある一方で、**自分が正常に意思決定できなくなることへの恐怖**がある。

この恐怖が、kiyomaroや経営ドキュメントを構築する**本質的な動機**になっている。「調子が良い日の自分」が判断基準を仕組み化しておくことで、「調子が悪い日の自分」が感情に流されず判断できる状態を作っている。

### 9.3. 経営判断への影響

- **判断すべきことの総量を減らす**ことを重視する。事業売却は利益確定であると同時に、認知負荷のカット
- **利益を確定してからフットワーク軽く動く**方を好む。不確定な新規事業に全財産を賭けるより、安全圏を確保してから複数のトライをする
- 判断の質を維持するために**オプショナリティ（選択肢の確保）**を重視する

## 10. このプロファイルの活用

### 10.1. 相談・提案時の原則

- **ダウンサイドシナリオを先に提示**すること。「最悪でもこうなる」が明確なら判断が速い
- **余計な修飾を排除**し、数字とオプションを端的に提示すること
- **推測ではなくデータソースを明示**すること。未確認の情報は「未確認」と明記
- **判断ポイントを増やす提案は避ける**。選択肢は絞って提示し、認知負荷を下げる

### 10.2. 特に注意すべきバイアス

- **安全圏での弛緩**: キャッシュが潤沢になったとき、「直接コミットしなくなる」兆候が出たら指摘すること
- **過大評価バイアス**: 新規事業で人に任せたいとき、「その人の能力の客観評価」と「自分が手を離したい動機」を分離するよう問いかけること
- **ビジョンとのズレ**: 規模拡大やトレンド追随の提案は池田さんのビジョンに合わない。「自由に前向きな仕事ができるか」「家族との時間を犠牲にしないか」が判断軸

### 10.3. 仕組み化の支援

- 「悲観的な日でも機能する判断基準」を一緒に設計することに最も価値がある
- 家族の安全が加わった判断軸を尊重する。以前と同じリスク許容度を前提にしない
- 売却後は「直接コミットの仕組み」（週次の数字レビュー等）を設計し、Innojin時代のパターン再発を防止する
