# Tetra LMS プラットフォーム事業計画

> **ステータス:** 開発中
> **最終更新:** 2026-03-19
> **売上実績:** 未開始（¥0）

---

## 1. 事業概要

| 項目 | 内容 |
|:---|:---|
| プロダクト | 研修コンテンツのマーケットプレイス型プラットフォーム |
| コンセプト | **Shopifyの研修版** — 研修事業者がTetra LMS上でコンテンツを販売 |
| 収益モデル | 売上の **10%** を手数料として徴収 |
| 技術基盤 | Tetra LMS（自社eラーニングと共通） |
| 位置づけ | 自社eラーニングの拡張。スケーラブルな収益源 |

### 自社eラーニング事業との関係

```
Tetra LMS（技術基盤：共通）
├── 自社eラーニング事業（business/e-learning.mdx）
│   └── SOCTが自社コンテンツを制作・販売
│       → 顧客: 障害福祉事業所
│       → 収益: ¥500/ユーザー/月
│
└── プラットフォーム事業（本ドキュメント）
    └── 外部の研修事業者がコンテンツを販売
        → 顧客: 研修事業者（コンテンツプロバイダー）
        → 収益: 売上の10%手数料
```

## 3. ビジネスモデル

### 3.1. 手数料構造

| 項目 | 内容 |
|:---|:---|
| 手数料率 | **10%**（売上に対して） |
| 決済 | プラットフォーム経由で集金 → 手数料差引後に事業者へ振込 |
| 価格設定 | 各事業者が自由に設定 |

### 3.2. 競合比較

| プラットフォーム | 手数料 | 特徴 |
|:---|---:|:---|
| Udemy | 37〜63% | 講師向け。集客力が高い |
| Teachable | 月額固定 | SaaS型。手数料なし |
| Shopify | 利用料+決済手数料 | EC特化 |
| **Tetra LMS** | **10%** | **福祉特化。Rakuco連携が差別化** |

> 10%は競合より低い水準。初期の事業者獲得を優先する価格設定。

| フェーズ | 研修事業者数 | 事業者あたり月次売上 | 手数料収入/月 |
|:---|---:|---:|---:|

| 拡大期（2028年〜） | 50 | ¥500,000 | ¥2,500,000 |

## 6. SOCTグループとのシナジー

| シナジー | 内容 |
|:---|:---|
| **Rakuco** | Rakuco利用事業所にプラットフォーム上の研修コンテンツを案内可能 |
| **はなえみダイレクト** | 企業の障害者雇用担当者向け研修コンテンツの販売チャネル |
| **自社eラーニング** | SOCTのコンテンツもプラットフォーム上で販売可能 |

## 7. リスク

| リスク | 深刻度 | 対策 |
|:---|:---|:---|
| 初期のコンテンツプロバイダー不足 | 🔴 | 研修事業者への営業、初期無料/低手数料期間の検討 |
| Rakucoと開発リソースの競合 | 🟡 | 自社eラーニング基盤との共通化で効率化 |
| 福祉業界以外への拡大判断 | 🟡 | まず福祉特化で実績を積む |
