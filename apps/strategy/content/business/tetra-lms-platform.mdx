## 1. 事業概要

| 項目 | 内容 |
|:---|:---|
| プロダクト | **ホワイトラベル型マルチテナントLMSプラットフォーム** |
| コンセプト | **Shopifyの研修版** — 研修事業者がTetra上で自社ブランドのLMSを運営し、自社の顧客企業に提供する |
| 収益モデル | **2層課金**（GMV × 10% + ストレージ利用料） |
| 技術基盤 | Next.js 16 / Supabase / Cloudflare Stream / BigQuery / Vercel |
| 位置づけ | SOCT事業ポートフォリオの**スケーラブルな収益エンジン** |

### 1.1. プロダクト定義

> **「教育コンテンツ」を「SaaSビジネス」に変換する、AI搭載型マルチテナントLMSエンジン」**

- **Mission**: 初期リスクゼロで始められる、本格的な自社ブランドLMS
- **Core Value**:
  - **B2B SaaS化**: コンテンツ配信だけでなく、サブスクリプション販売・顧客管理・組織管理をオールインワンで提供
  - **Risk Free**: テナントは固定費ゼロ（ストレージスターターは無料）で開始可能
  - **AI-Powered**: Gemini 2.5を活用した学習アシスタント機能で差別化（Phase 16予定）

### 1.2. 4層テナントモデル

```
L0: Platform（Tetra = SOCT）
  └── L1: Tenant（教育事業者 = SaaS利用者/販売者）
        ├── 自社ブランドのLMSとして運営
        ├── コース作成・CMS管理
        ├── 料金設定（seat_price / min_seats）
        ├── Stripe Connect で売上を直接受け取る
        └── L2: Organization（テナントの顧客企業）
              ├── セルフサーブ登録（CSコスト不要）
              ├── Trial → Active → Suspended のライフサイクル
              └── L3: User（受講者 = 顧客企業の従業員）
                    └── 組織コード + ログインIDで認証
```

| レイヤー | 役割 | 具体例 |
|:---|:---|:---|
| **Platform** | Tetra全体の運営 | SOCT |
| **Tenant** | 教育コンテンツを販売する事業者 | はなえみラーニング（SOCT）、保育園コンサル会社 |
| **Organization** | テナントの顧客企業 | 障害福祉事業所、保育園 |
| **User** | 実際に学習する従業員 | 事業所の支援員、保育士 |

### 1.3. UX/UIポリシー

**「無色透明」かつ「実務的」なモダンビジネスUI**

- 特定の業界色（福祉、医療、IT等）を持たせない
- Slack / Notion / Linear等のモダンSaaSと同様の機能美を追求
- エンドユーザーが「Tetra」を意識しないホワイトラベル体験

## 3. ビジネスモデル

### 3.1. 2層課金構造

```
┌─────────────────────────────────────────────────────┐
│  Layer 1: 組織 → テナント → Tetra（取引手数料）      │
│  ─────────────────────────────────────────────────── │
│  組織が支払うLMS利用料 → Stripe Connect経由で         │
│  テナントに直接送金 → Tetraが13.6%を天引き           │
│  （内訳: Tetra 10% + Stripe決済手数料 3.6%）         │
├─────────────────────────────────────────────────────┤
│  Layer 2: テナント → Tetra（ストレージ利用料）        │
│  ─────────────────────────────────────────────────── │
│  動画保存容量に応じた月額固定費                        │
│  Stripe Billingで直接課金                             │
└─────────────────────────────────────────────────────┘
```

### 3.2. Layer 1: 取引手数料

| 項目 | 内容 |
|:---|:---|
| **課金単位** | `seat_price × max(active_users, min_seats)` / 月 |
| **課金方式** | Stripe Connect (Destination Charge) |
| **Tetra手数料** | GMV × 10% |
| **決済手数料** | GMV × 3.6%（テナント負担。Stripe処理費） |
| **決済方法** | カード決済（前払い/自動） / 銀行振込（当月払い/月末締め） |
| **日割り精算** | ユーザー増減時はStripe日割り計算を自動適用 |

- テナントが `seat_price`（1ユーザーあたり月額単価）と `min_seats`（最低契約ユーザー数）を自由に設定
- 組織（テナントの顧客企業）は**セルフサーブ登録**（テナント側のCS不要）
- Trial期間（0〜30日）はテナントが設定可能

### 3.3. Layer 2: ストレージ利用料

| プラン | 動画保存枠 | 月額 |
|:---|:---|---:|
| **スターター** | 5時間 | ¥0 |
| ライト | 20時間 | ¥1,500 |
| スタンダード | 50時間 | ¥3,800 |
| プロ | 100時間 | ¥6,800 |

> テナントは無料から始められる。コンテンツが増えるにつれて自然にアップグレードする構造。

### 3.4. 手数料体系サマリ（テナント負担）

| # | 名目 | 計算式 | 備考 |
|:---|:---|:---|:---|
| ① | プラットフォーム利用料 | GMV × 10% | Tetra収益 |
| ② | 決済手数料 | GMV × 3.6% + ¥30/決済 | Stripe処理費（¥30はTetra吸収） |
| ③ | 入金手数料 | 入金額 × 0.25% + ¥450/回 | Stripe振込処理費 |

> **テナントの受取額** = GMV − ①プラットフォーム利用料 − ②決済手数料 − ③入金手数料

### 3.5. 競合比較

| プラットフォーム | 手数料 / 月額 | 特徴 | Tetraの優位性 |
|:---|:---|:---|:---|
| **Teachable** | $0〜$665/月（固定費） | コース販売PF。B2C寄り | 固定費不要。B2B特化 |
| **Thinkific** | $0〜$499/月（固定費） | 同上 | 同上 |
| **LearnWorlds** | $29〜$299/月（固定費） | ホワイトラベルLMS | GMV連動で低リスク |
| **Udemy Business** | 37〜63%の手数料 | 集客力は高いが自社顧客は持てない | 自社顧客を保有できる |
| **Tetra LMS** | **GMV × 10% + ストレージ** | **福祉/保育特化。セルフサーブ。AI搭載予定** | — |

> Tetraの差別化: 固定費ゼロ（GMV連動）でリスクフリー + セルフサーブによるCS不要構造 + 日本語完全対応 + 業界特化。

## 4. eラーニング事業との関係

### 4.1. 構造

```
Tetra LMS プラットフォーム（本ドキュメント）
│
├── テナント#1: はなえみラーニング（SOCT自社事業）
│   ├── 4月〜 βテスト、7月〜 課金開始
│   ├── 顧客: 障害福祉事業所（Rakuco顧客中心）
│   ├── seat_price: ¥800/ユーザー/月（Rakuco特典: 初年度¥400）
│   └── 詳細: → e-learning.mdx
│
├── テナント#2: 保育園向けコンサル会社（商談中）
│   ├── 7月以降のローンチを想定
│   ├── 顧客: 保育園・こども園 約50社
│   └── seat_price: ¥1,000/ユーザー/月（想定）
│
└── テナント#3〜: 今後の営業次第
```

### 4.2. 事業会計上の区分

| | eラーニング事業 | プラットフォーム事業 |
|:---|:---|:---|
| **収益源** | 組織からの利用料（seat_price × users） | テナントからの手数料（GMV × 10%）+ ストレージ料 |
| **SOCT内の位置** | 事業部門（コンテンツ制作・営業） | インフラ部門（プラットフォーム開発・運営） |
| **計上** | はなえみラーニングの売上として計上 | プラットフォーム手数料として別計上 |

> はなえみラーニングがTetraの最初のテナントとして運用実績を積む → ショーケースとして外部テナント獲得の営業に活用。

### 4.3. 自社テナント（はなえみラーニング）の手数料について

はなえみラーニングはSOCTが運営するテナントのため、手数料10%はSOCT内で循環する。**外部テナントが加わって初めてプラットフォーム事業としての独立収益が発生する。**
