# ヒアリングシート（初回商談用）v4.0

> **最終更新:** 2026-04-21
> **位置づけ:** 初回商談で使用する**データ収集フォーム**。ここで得た情報 + ワムネット公開データを統合して「運営分析レポート」を作成する。
> **参照元:** [コンサル実務ガイド](./consulting-ops-guide) / [コンサルティング事業計画](../business/consulting)

---

## 商談前の準備

> ヒアリング前にワムネットで取得済みのデータを確認し、**既に分かっていることは聞かない**。

- [ ] ワムネット事前調査が完了している
- [ ] 事業所の種別（A型 / B型 / 両方）を把握している
- [ ] 加算の取得状況を一覧化している
- [ ] 従業者情報（人数・資格）を確認している
- [ ] A型の場合：公表スコアを確認している

---

## 商談の基本スタンス

- **「教える人」ではなく「一緒に整理する人」。** 現場のことは経営者が一番知っている。我々はデータと制度の知見で補完する。
- **ヒアリングの目的を最初に伝える。** 「御社の状況を詳しく伺い、攻め・守り・経営基盤の3面で分析レポートをお作りします」
- **この場で答え（シミュレーション・数字の約束）は出さない。** 「次回、分析結果をお持ちします」

---

## §1. アイスブレイク（5分）

**目的:** 関係構築 + ヒアリングの目的説明

**話すこと:**
- 自己紹介（はなえみ × SOCT の体制）
- 「御社の情報を事前に調べさせていただきました。本日は、公開データだけでは分からない御社の実態を伺い、**現状の診断レポートを作成して次回お持ちしたい**と考えています」
- 「攻め（増収の可能性）・守り（リスク回避）・経営基盤の3つの観点でお聞きします」

---

## §2. 共通ヒアリング項目（A型・B型とも）（30分）

> **健康診断のイメージ。** 全クライアントに対して同じ検査項目で診断する。漏れなく確認していく。

### 2.1. 加算の取得状況

> ワムネットで取得済み加算は確認済み。ここでは**取得できていない理由**と**取得可能性**を聞く。

- [ ] 現在取得している加算の一覧を把握しているか（経営者自身の認識）
- [ ] 取得を検討したが断念した加算はあるか → その理由（事務負担 / ルール不明 / 要件不足）
- [ ] 有資格者（社会福祉士・精神保健福祉士等）がいるのに加算に紐づけていないケースはないか
- [ ] 処遇改善加算の取得区分（I〜IV）と、上位区分への移行を検討しているか
- [ ] 処遇改善加算の計画書の作成・提出は誰が担当しているか
- [ ] 「見える化」（職場環境等要件の公表）は対応済みか

### 2.2. 減算リスクの対応状況

> **ここが守りの核心。** 「未対応なら報酬返還リスクがある」ことを共有する。

**BCP（業務継続計画）:**
- [ ] 感染症BCPは策定済みか
- [ ] 自然災害BCPは策定済みか
- [ ] 策定してから見直し（更新）はしているか
- [ ] BCPに基づく研修・訓練は実施しているか

**情報公表（ワムネット）:**
- [ ] 直近の報告期限を把握しているか
- [ ] 報告内容は最新の状態か
- [ ] 誰が報告作業を担当しているか

**虐待防止:**
- [ ] 虐待防止委員会は設置しているか
- [ ] 年1回以上の研修を実施しているか
- [ ] 担当者は配置しているか

**人員配置基準:**
- [ ] 出勤簿と支援記録が1日単位で一致しているか
- [ ] 突合チェックは誰が、どの頻度で行っているか
- [ ] 人員基準を下回るリスクのある日はあるか（欠勤時の体制）

### 2.3. 売上・稼働率

> 「人を入れる」「売上を立てる」の両面を確認。

**利用者の確保（集客）:**
- [ ] 現在の利用者数と定員の比率（ワムネット確認済みの数字と実態の差異）
- [ ] 新規利用者の獲得チャネル（相談支援事業所 / HP / 口コミ / 行政紹介）
- [ ] 相談支援事業所への営業活動はしているか → 頻度と方法
- [ ] 集客に課題を感じているか → 具体的にどの部分が難しいか
- [ ] 直近3ヶ月の新規利用者数と退所者数

**稼働率:**
- [ ] 平均出席率（＝稼働率）は何%くらいか
- [ ] 出席率の把握はしているか（日次 / 月次 / していない）
- [ ] 出席率が低い利用者への対応（連絡体制、通所促進）

**売上の構成（A型の場合は生産活動収支で別途確認）:**
- [ ] メインの作業・仕事の内容
- [ ] 仕事の確保に課題はあるか（受注先の開拓等）

### 2.4. 人材・定着

> 「人を入れる + 辞めさせない」の両面を確認。

**人員配置:**
- [ ] 現在の人員配置で余裕はあるか
- [ ] サービス管理責任者（サビ管）は何名配置しているか
- [ ] 管理者とサビ管の兼務はあるか
- [ ] 有資格者（社会福祉士等）で、加算に紐づけていない職員はいないか

**採用:**
- [ ] 採用に困っているポジションはあるか
- [ ] 直近1年で何名採用し、何名退職したか
- [ ] 採用チャネル（求人媒体 / ハローワーク / 紹介等）

**定着:**
- [ ] 離職率は高いと感じているか
- [ ] 退職の主な理由は把握しているか
- [ ] 定着のための施策はあるか（ボーナス / 福利厚生 / 研修 / キャリアパス等）
- [ ] 職員の満足度を把握する仕組みはあるか（面談 / アンケート等）

### 2.5. 社内情報の透明化・ガバナンス

> **議事録分析で完全に欠落していた観点。** 経営者が現場の実態を把握できているかを確認する。

**会議・報告体制:**
- [ ] 社内の定例会議はあるか → 頻度と参加者
- [ ] 経営者は毎回会議に参加しているか（「毎回俺が参加してる」状態か）
- [ ] 会議の議事録は取っているか → 方法（紙 / ノート / デジタル）
- [ ] 議事録は経営者まで共有されているか

**現場の状況把握:**
- [ ] 利用者とのトラブル・苦情は経営者まで報告が上がる仕組みがあるか
- [ ] 現場の職員に業務を任せきりにしている部分はどこか
- [ ] 「任せているが心配」な領域はあるか

**情報共有:**
- [ ] 職員間の情報共有の方法（朝礼 / 引き継ぎノート / チャットツール）
- [ ] ヒヤリハット・事故報告の記録と共有体制

### 2.6. ITツール・業務効率化

> 具体的なツール名と導入レベルを確認する。DX提案の余地を探る。

**記録・請求:**
- [ ] 記録・請求ソフトは何を使っているか（Rakuco / 他社 / Excel / 紙）
- [ ] 記録の入力は誰が、いつ行っているか（リアルタイム / 翌日まとめ）
- [ ] 国保連への請求作業は誰が担当しているか

**コミュニケーション:**
- [ ] 社内の連絡手段（LINE / メール / 電話 / チャットツール）
- [ ] 外部（利用者家族・相談支援事業所）との連絡手段

**自動化・効率化:**
- [ ] 議事録の自動化（文字起こし等）に関心はあるか
- [ ] 業務の中で「もっと効率化できたら」と感じている作業はあるか
- [ ] HPは持っているか → 更新頻度と担当者

### 2.7. 経営者のビジョンとリソース

> 「経営者が本来やりたいこと」と「実態」のギャップを確認。

- [ ] 経営者が事業所を始めた理由・ビジョン
- [ ] 現在、経営者の時間の大半を使っている業務は何か
- [ ] 「本来やりたいが手が回っていないこと」は何か
- [ ] 事業拡大（多店舗展開等）の意向はあるか

---

## §3. A型専用ヒアリング項目（10分）

> A型でない場合はスキップ。

### 3.1. スコア制の状況

- [ ] 現在のスコア（自己認識）と目標スコア
- [ ] 新設3項目（支援力向上・経営改善計画・知識能力向上）への対応状況
- [ ] スコアを上げるために取り組んでいること

### 3.2. 生産活動収支

- [ ] 直近の生産活動収支は黒字か赤字か
- [ ] 赤字の場合、経営改善計画の提出状況
- [ ] 主な売上の柱となっている作業・仕事

### 3.3. 最低賃金対応

- [ ] 最低賃金改定時の対応方針
- [ ] 人件費増への対策（作業単価アップ・効率化等）

---

## §4. B型専用ヒアリング項目（10分）

> B型でない場合はスキップ。

### 4.1. 工賃関連

- [ ] 現在の平均工賃額
- [ ] 工賃向上計画は策定しているか
- [ ] 目標工賃達成加算の存在を知っているか

### 4.2. 指導員配置

- [ ] 工賃向上の専門指導員は配置しているか
- [ ] 目標工賃達成指導員配置加算を申請しているか

### 4.3. 利用時間

- [ ] 平均利用時間はどれくらいか
- [ ] 4時間未満の利用者の割合（短時間利用減算リスクの確認）

---

## §5. 次回アクションの確定（5分）

**話すこと:**
- 「本日お伺いした内容と事前の調査データを統合して、**御社専用の運営分析レポート**を作成します」
- 「攻め・守り・経営基盤の3面で御社の現状を整理し、次回お見せします」
- **この場で具体的な改善策や数字の約束はしない**

| 確定すること | 詳細 |
|:---|:---|
| **次回日程** | 「分析レポートのご説明」として1〜2週間後を目安に確定 |
| **追加データの依頼**（必要な場合） | 出勤率の詳細データ、工賃の実績値等 |

> ⚠️ **注意:**
> - 商談中は北澤の判断を介さずに「この加算が取れます」等の断定はしない
> - 具体的なアクションプラン・テンプレート提供は契約後のサービス
> - [法的ガードレール](./consulting-ops-guide#6-️-法的ガードレール--絶対に守ること) を遵守
