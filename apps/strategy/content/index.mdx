# 🏯 Kiyomaro 経営戦略ドキュメント

株式会社SOCTの経営戦略ドキュメントを管理するためのサイトです。

## 概要

本サイトでは、以下の経営戦略ドキュメントをバージョン管理しています。

- **中期経営計画** — 年度ごとの成長戦略と財務計画
- **事業戦略** — 各事業セグメントの詳細な戦略と施策
- **組織戦略** — 人材・組織に関する方針（今後追加予定）

## 運用ルール

| 項目 | ルール |
|:---|:---|
| **ドキュメント形式** | Markdown（`.mdx`） |
| **バージョン管理** | Git（PR ベースのレビュー・承認フロー） |
| **AI 提案** | Kiyomaro が PR として改訂案を提出 |
| **承認** | 経営陣によるレビュー → マージ |
| **履歴** | Git ログによる完全な監査証跡 |

## コア設計思想

1. **Human-in-the-loop**: AI が戦略を書き換えることはない。常に「提案」し、人間が「承認」する
2. **完全な監査証跡**: すべての変更履歴を Git で永続保持
3. **段階的な進化**: 戦略の変遷を時系列で辿ることが可能
